### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Anareony/Only.git
   ```
2. Navigate to the project directory
   ```sh
   cd only
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Запустите проект
   ```sh
   npm start
   ```
5. The project will run on
   ```sh
   http://localhost:3000
   ```
